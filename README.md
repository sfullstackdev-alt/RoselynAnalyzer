# Roselyn Analyzer

A Python analysis tool.

## Installation

```bash
pip install -e .
```

For development:

```bash
pip install -e ".[dev]"
```

## Usage

```bash
roselyn
```

Or as a module:

```python
from roselyn_analyzer import main
main()
```

## Development

Run tests:

```bash
pytest
```

Format code:

```bash
black src tests
ruff check src tests
```
